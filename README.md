# Electric Vibraphone

  * ArduinoとPureDataによる電子ビブラフォン
  * ![](http://gyazo.com/b871d407db9a370f7d5939f129689b36.png)

## 必要な物をインストール

  * Arduino IDE
    * http://arduino.cc/en/Main/Software
  * PureData
    * http://puredata.info/downloads/pd-extended
  

## Arduinoを接続
  
  * Arduino Micro または Arduino Leonard (ATmega32u4搭載のもの)
  	* main.ino を書き込む

  * ピエゾ素子
    * A0~A5,8,9に繋ぐ
    * 500kΩのカーボン抵抗も繋ぐ
