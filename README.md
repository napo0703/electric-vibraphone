# Electric Vibraphone

  * ArduinoとPureDataによる電子ビブラフォン

## 必要な物をインストール

  * Arduino IDE
    * [http://arduino.cc/en/Main/Software]
  * PureData
    * [http://puredata.info/downloads/pd-extended]
  

## Arduinoを接続
  
  * Arduino micro または Arduino leonard (atmega32u4搭載のもの)
	* main.ino を書き込む

  * ピエゾ素子
    * A0~A5,8,9に繋ぐ
    * 500kΩのカーボン抵抗も繋ぐ
